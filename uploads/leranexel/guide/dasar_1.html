Pengertian, Fungsi, Sejarah dan Versi Microsoft Excel

Banyak dari kita mungkin sudah tidak asing lagi ketika mendengar Microsoft Excel. Aplikasi ini sering digunakan oleh berbagai kalangan mulai dari siswa, mahasiswa maupun pekerja karena sangat bermanfaat untuk mengolah angka.

MS Excel sebenarnya telah hadir sejak lama dan hingga sekarang mempunyai banyak versi yang sudah dikeluarkan. Microsoft sendiri dari waktu ke waktu masih mengembangkan aplikasi tersebut agar kinerjanya semakin baik dengan fitur yang semakin canggih. Sebelum belajar Microsoft Excel secara praktik, mari kita bahas materi ini terlebih dahulu agar kita lebih mengenal tentang aplikasi ini.

Pengertian Microsoft Excel

Microsoft Excel merupakan program aplikasi spreadsheet pengolah angka yang dikembangkan oleh Microsoft digunakan untuk mengolah data dalam bentuk baris dan kolom.

Software aplikasi ini biasa dimanfaatkan dalam mengolah data yang membutuhkan perhitungan karena memiliki fitur-fitur khusus baik itu berbentuk formula (rumus) atau fitur pendukung yang lain. Berbeda dengan aplikasi pengolah angka lainnya, MS Excel termasuk aplikasi spreadsheet yang premium atau berbayar. Kita harus membeli lisensi agar dapat menggunakannya secara resmi.

Pada versi PC atau laptop, Microsoft Excel tersedia untuk sistem operasi Windows dan macOS. Sedangkan di versi HP, aplikasi ini tersedia untuk Android dan iPhone namun dengan fitur yang tidak selengkap versi PC-nya. Untuk versi HP, aplikasi ini dapat digunakan secara gratis.

Fungsi Microsoft Excel

Fungsi utama program Microsoft Excel adalah untuk mengolah data berupa angka. Selain fungsi utama tersebut aplikasi ini juga memiliki fungsi dan kegunaan lainnya, diantaranya adalah:

  • Untuk membuat, mengolah dan menyimpan data.
  • Untuk membuat grafik atau diagram dari data yang dimasukkan.
  • Untuk menyelesaikan perhitungan statistika, aritmatika dan logika.
  • Sebagai aplikasi akuntan untuk merekap laporan keuangan.
  • Sebagai alat untuk melakukan analisa data.
  • Untuk melakukan perhitungan keuangan seperti kurs mata uang.
  • Sebagai database sederhana.

Sejarah Microsoft Excel

Microsoft meluncurkan program aplikasi pertama Microsoft Excel secara resmi pada tahun 1985 untuk sistem operasi Macintosh. Sedangkan aplikasi MS Excel untuk Windows resmi dirilis pada tahun 1987. Saat itu versi MS Excel sudah memasuki versi yang kedua (2.0).

Sebelum adanya aplikasi Ms Excel kebanyakan orang saat itu telah mengenal program aplikasi spreadsheet bernama Lotus. Aplikasi Lotus sudah ada sejak tahun 1982 hingga akhirnya Microsoft membuat aplikasi Ms Excel yang mampu menjadi pesaingnya. MS Excel mampu mengalahkan Lotus di era 90-an seiring dengan dirilisnya versi 3.0 pada tahun 1990. Lalu disusul versi 4.0 pada tahun 1992 dan seterusnya.

Sejak saat itu Microsoft terus mengembangkan aplikasi MS Excel dan hampir setiap dua tahun sekali Microsoft merilis versi baru dengan menyajikan fitur yang baru. Sampai sekarang Microsoft masih terlihat konsisten untuk mengembangkan versi-versi terbarunya. Sampai Saat ini Microsoft telah merilis versi terakhirnya yaitu MS Excel 2021
