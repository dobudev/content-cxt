Shortcut excel adalah pintasan keyboard microsoft excel yang digunakan sebagai alternatif lain untuk mengakses atau mengaktifkan perintah-perintah pada Microsoft Excel tanpa menggunakan tetikus(mouse).

Dengan Shortcut di excel berikut anda dapat menggunakan kombinasi beberapa tombol pada papan ketik atau keyboard untuk melakukan aktifitas tertentu pada excel.

Shortcut Excel

Saat menggunakan microsoft excel, Anda dapat menekan tombol Alt atau F10 untuk mengaktifkan Key Tips sehingga muncul huruf-huruf tertentu di sebelah TAB ribbon dan Quick Access Toolbar.


Jika Anda menekan salah satu huruf yang sesuai pada keyboard, maka tindakan tersebut akan mengaktifkan TAB atau perintah pada QAT.

Apabila Anda memilih mengaktifkan salah satu TAB pada ribbon, maka akan muncul Key Tips pada masing-masing tombol perintah untuk mengaktifkannya.

Misalnya setelah Anda menekan tombol Alt atau F10, kemudian Anda memilih menekan tombol M untuk mengaktifkan TAB Formula. Selanjutnya untuk mengaktifkan kotak dialog Name manager tekan tombol N.



Kombinasi Shortcut Keyboard dengan key tips ini bisa di tekan secara bergantian. Berbeda dengan daftar kombinasi Shortcut excel di bawah yang harus Anda tekan bersamaan/sekaligus.

Jika anda sudah terbiasa menggunakan shortcut-shortcut excel ini tentunya akan menghemat waktu sehingga pekerjaan anda akan lebih cepat rampung.

Pada dasarnya kebanyakan shortcut excel berikut juga bisa digunakan pada aplikasi lain untuk beberapa fungsi umum semisal perintah copy, paste, print, dan lain sebagainya. Jadi Anda tidak perlu lagi ragu untuk mencoba dan mulai membiasakannya.
