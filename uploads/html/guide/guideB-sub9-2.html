Mengatur Ukuran Border CSS

Untuk mengatur ukuran pada border kita bisa menggunakan property css yaitu border-width.berikut ini adalah contoh cara penulisan property border-width untuk membuat ukuran garis.

index.html


style.css

#garis_1{    border-style:solid;    border-widht:5px; } #garis_2{    border-style:dotted;    border-widht:10px; } #garis_3{    border-style:dashed;    border-widht:3px; } #garis_4{    border-style:double;    border-widht:9px; } #garis_5{    border-style:groove;    border-widht:25px; } #garis_6{    border-style:inset;    border-widht:5px; } #garis_7{    border-style:outset;    border-widht:5px; } #garis_8{    border-style:ridge;    border-widht:50px; }

dan perhatikan ukuran garis setelah kita mengaturnya dengan border-width.
